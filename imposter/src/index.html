Local Development Instructions

This project is a partial component of an Imposter game, in which there are several players who know a certain category and every player except one (the imposter) gets the word in that category. Players take turns telling clues in such a way so as to make sure the imposter isn't able to guess the word at the end of the round. Players vote on who they think the imposter is at the end of the round. This project displays a random word and a random category. My hope is that others will build off of this to complete the Imposter game. Users can add categories and words to the Categories.txt file in the assets folder. However, the code currently only works if there are one thousand words in each category, the category title has "Category:" before it, and there are new lines after each line of text. After downloading, go the command line and change the directory to the imposter (not Imposter) directory. Make sure you have node package manager installed. Run the command "npm run dev" to run the project.
